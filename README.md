# Claude Skills

Reusable skills for Claude Code. Each skill defines a workflow, team structure, or automation pattern.

## Skills

### `hackathon.md`
Structured development sessions with pod-based feature delivery. Coordinator + Product Manager + 4-member feature pods (Implementer, Validator, Meta-Validator, Deslopper) with mandatory sign-off before integration.

## Usage

Copy a skill to your project's `.claude/skills/` directory, or reference it from your Claude Code configuration.
